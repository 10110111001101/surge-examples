# Bathymetry Coverage

Global strip of bathymetry for latitudes [-50, 50] broken up in 45 degree 
increments starting at -180 (the anti-meridion).  Files are enumerated 
starting with longitude [-180, -135] and counting up such that:

strip_1.nc - [-180, -135]
strip_2.nc - [-135,  -90]
strip_3.nc - [ -90,  -45]
strip_4.nc - [ -45,    0]
strip_5.nc - [   0,   45]
strip_6.nc - [  45,   90]
strip_7.nc - [  90,  135]
strip_8.nc - [ 135,  180]
